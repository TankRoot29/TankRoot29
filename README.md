### 📡 Projets Réseaux & IoT

- 🅿️ **[IoT Smart Parking & Edge Security](https://github.com/TankRoot29/Projet-IoT---parking-iot-service.git)** ⭐ *Projet phare*
  Plateforme IoT distribuée : capteurs ESP32 (HC-SR04, DHT22, LCD, servo), Edge IDS embarqué pour la détection d'intrusions en temps réel, pipeline `MQTT → Node-RED → InfluxDB → Grafana`.
  `ESP32` `MQTT` `Node-RED` `InfluxDB` `Grafana` `Docker`

- ⚖️ **[Load Balancing DNS & Web](https://github.com/TankRoot29/Projet-LoadBalancing.git)**
  Infrastructure haute disponibilité : cluster DNS (BIND9) + ferme web (Apache2) répartis via iptables DNAT/NAT, émulés sous Kathara/Docker. Répartition ~50/50 validée sur 100 requêtes.
  `Kathara` `iptables` `BIND9` `Apache2` `Docker`

- 🎧 **[AudioStream UDP](https://github.com/TankRoot29/Diffusion-de-flux-Audio-avec-UDP.git)**
  Streaming audio WAV temps réel en Java, jitter buffer intelligent (`ConcurrentSkipListMap`), réordonnancement des paquets, stats live (latence, pertes).
  `Java` `UDP` `Java Sound API` `Jitter Buffer`
